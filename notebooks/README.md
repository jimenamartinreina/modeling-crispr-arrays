# Jupyter Notebooks for Simulation Analysis

This directory contains Jupyter notebooks written in Python to analyze simulation results. The simulations explore different array sizes and parameter configurations.

## Folder Structure

- `analysis_nsp10`: Analysis for simulations with array size = 10 spacers.
- `analysis_nsp40`: Analysis for simulations with array size = 40 spacers.
- `analysis_nsp40_g60`: Analysis for array size = 40 spacers with a modified `g` parameter, resulting in different curve morphologies.

## Simulation Scripts

- For `analysis_nsp10` and `analysis_nsp40`, simulations were generated using the C++ script `simulate_spacers1.2`.
- For `analysis_nsp40_g60`, simulations were generated using `simulate_spacers1.3`.
