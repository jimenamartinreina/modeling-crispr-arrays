# Parameter Handling for Massive Simulations

This folder contains scripts and resources used to execute massive simulations by generating and managing parameter combinations.

## Contents

- **`parameters_table_1.2.py`**:  
    This script generates combinations of different parameter values and outputs a `.txt` file containing a list of all possible parameter combinations. This file serves as input for running simulations.

- **`run_simulations.sh`**:  
    A shell script that automates the execution of simulations. It takes the parameter combinations file generated by `parameters_table_1.2.py` and the script in `main_simulations/simulate_spacers` (any version) to run all simulations in sequence.

## Workflow

1. Use `parameters_table_1.2.py` to create a `.txt` file with all desired parameter combinations.
2. Place the generated `.txt` file in the same location as `simulate_spacers.cpp` and `run_simulations.sh` scripts.
3. Modify `run_simulations.sh` script accordingly to your desired input and output files/directories.
4. Execute `run_simulations.sh` to run all simulations.
5. See the txt. output files in the folder you defined in `run_simulations.sh` script.

## Requirements

- Python (for running `parameters_table_1.2.py`)
- Bash (for executing `run_simulations.sh`)
- C++ (for running `simulate_spacers.cpp`)

## Notes

- Modify the scripts as needed to fit your specific simulation requirements.
