# Time Simulation Analysis

This folder contains resources for visualizing and analyzing simulation outputs that differ in time, using an arbitrary set of input parameters.

## Contents

- **Jupyter Notebook**  
    Provides interactive visualization of simulation results. Use this notebook to explore and compare outputs from different simulation runs.

- **Data Folder**  
    Contains the results of simulations used in the notebook. These were performed with the C++ script `simulate_spacers1.1`.

## Notes

- All simulation outputs were generated using `simulate_spacers1.1` (C++ script).
- The notebook is designed to work with the data in the data folder.
